Звіт про виконання Лабораторної Роботи № 05 (ООП)

Виконавець: Лукіян О.В.
Тема: Інкапсуляція.

---
ВИКОНАНІ ЗАВДАННЯ:

1. Ферма (Animal Farm):
   - Мета: Створення класу Chicken з прихованими даними.
   - Концепції: Інкапсуляція полів (private), використання властивостей (Name, Age) для контрольованого доступу. Реалізована валідація даних у сетерах.

2. Похід за покупками (Shopping Spree):
   - Мета: Моделювання взаємодії покупців (Person) та товарів (Product).
   - Концепції: Інкапсуляція логіки транзакцій (BuyProduct) всередині класу Person. Використання колекції List<Product> для "сумки". Реалізована валідація на відсутність від'ємних значень (гроші, вартість) та на порожні імена.

3. Калорійність піци (Pizza Calories):
   - Мета: Створення складної композиції класів (Pizza, Dough, Topping) з багаторівневою валідацією.
   - Концепції: Композиція, де Pizza інкапсулює Dough та List<Topping>. Реалізована сувора валідація на допустимі типи (борошно, начинки) та діапазони ваги/кількості. Обчислювальний метод TotalCalories() для агрегованого розрахунку.

---
РЕЗУЛЬТАТ:
Усі 3 завдання успішно виконано. Продемонстровано застосування принципу інкапсуляції для створення стійких та валідованих класів, що захищають внутрішній стан об'єктів та забезпечують цілісність даних.
